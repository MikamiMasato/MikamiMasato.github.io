三上雅人のGitHub Pages

はじめに

三上雅人(みかみまさと)と申します。 アニメの演出をしています。

Git と GitHub について学んでいたところ、GitHub Pages という便利な機能があることを知りまして、 学習ついでにひとつやってみようと思ったのと、ネット上に偽情報やあやふやな情報が溢れる昨今ですので、 正しい情報をまとめておこうと思い作りました。
